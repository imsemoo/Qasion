# قالب مركز قاسيون للدراسات الاستراتيجية — Qasioun Theme

قالب موقع أخبار/أبحاث ثابت (Static) بلغة **HTML / CSS / JS** خالص، بدون أي إطار عمل (framework) أو أدوات بناء (build tools). عربي بالكامل (RTL).

تم تحويله من نموذج تصميم (Claude Design) كان يعتمد على runtime خاص، إلى قالب نظيف قابل للاستخدام والتعديل مباشرة.

## التشغيل

القالب ثابت — يكفي فتحه عبر أي خادم ملفات بسيط (وليس `file://` مباشرةً لأن الخطوط والصور الخارجية تحتاج `http`).

```bash
# داخل مجلد qasioun-theme
npx serve .
# أو أي خادم static آخر، ثم افتح http://localhost:3000
```

## مبادئ التنظيم

- **لا CSS داخل HTML** — كل التنسيق في ملفات `css/` عبر أصناف (classes). (الاستثناء الوحيد: عرض أشرطة البيانات في إنفوغرافيك الصفحة الرئيسية `style="width:%"` لأنها قيمة بيانات.)
- **الهيدر والفوتر مصدر واحد** في `js/layout.js` — عدّل مرة واحدة تنطبق على كل الصفحات. حدّد الصفحة النشطة عبر `data-page` على `<body>`.
- **متغيّرات التصميم (Design tokens)** في `:root` داخل `css/base.css` — الألوان والخطوط والمقاسات في مكان واحد.
- **responsive** — تتجاوب الصفحات حتى شاشات الجوال مع قائمة تنقّل منسدلة.

## التخصيص

- الألوان/الخطوط: عدّل المتغيّرات في أعلى `css/base.css`.
- روابط/عناصر التنقّل: مصفوفة `NAV` في `js/layout.js`.
- البيانات: `js/data/publications.js` (استبدلها بمصدر حقيقي لاحقاً).
- الخطوط وSwiper تُحمَّل من CDN؛ يمكن استضافتها محلياً عند الحاجة.
